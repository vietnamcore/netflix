# Netflix Premium Landing Page

## Giới thiệu
Landing page cao cấp với phong cách Netflix Premium, tích hợp hiệu ứng 3D, glassmorphism và animations chuyên nghiệp.

## Tính năng
- 🎬 Hero với 3D Laptop, TV, Phone, Popcorn
- 💳 Bảng giá với 3 gói dịch vụ
- 🎨 Glassmorphism và hiệu ứng Cyber
- 🔮 Hiệu ứng 3D, Particle, Aurora
- 📱 Responsive 100%
- ⚡ Hiệu suất tối ưu

## Công nghệ sử dụng
- HTML5, CSS3, JavaScript ES6
- GSAP cho animations
- Three.js cho 3D objects
- AOS cho scroll animations
- Particles.js cho hiệu ứng hạt
- Lenis cho smooth scroll

## Cài đặt

### Yêu cầu
- Node.js (phiên bản 14+)
- npm

### Các bước
```bash
# Clone repository
git clone https://github.com/yourusername/netflix-premium.git

# Di chuyển vào thư mục
cd netflix-premium

# Cài đặt dependencies
npm install

# Chạy development server
npm run dev
